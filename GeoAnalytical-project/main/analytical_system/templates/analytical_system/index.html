Возможности Модуля Анализа

Модуль анализа больших данных - одна из частей комплекса программ ГИС Горизонт, которая способна обрабатывать пространственные данные из хранилища, рассчитывать множество показателей, визуализировать и создавать отчеты.

Скорость обработки одного миллиона строк данных из таблиц базы данных Postgres достигает 8,34 секунд

Способы Визуализации Данных

Данный модуль способен представлять большие данные в виде, который обеспечивает наиболее эффективную работу человека по их изучению и пониманию. Данная система визуализирует данные в различных форматах в зависимости от выбора пользователя:

  • таблица;
  • линейная диаграмма;
  • гистограмма;
  • географическая диаграмма.

Эффективность Системы

Вся система написана на языке программирования Python, которая в свою очередь взаимодействует, как с хранилищем пространственных данных ГИС Горизонт, так и со своей базой данных Postgres. Все рассчеты и манипуляции с данными происходят так быстро благодаря библиотека Python — Pandas. Она необходима для обработки и анализа структурированных данных, её название происходит от «panel data» («панельные данные»). Панельными данными называют информацию, полученную в результате исследований и структурированную в виде таблиц.
